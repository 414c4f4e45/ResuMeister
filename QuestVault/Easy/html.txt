Q: What is HTML?
A: HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages. HTML provides the basic structure of a webpage using elements like headings, paragraphs, links, images, and other types of content.

Q: What is the purpose of the <!DOCTYPE html> declaration?
A: The <!DOCTYPE html> declaration is used to specify the document type and version of HTML being used. It helps the web browser render the page correctly by defining that the document is written in HTML5.

Q: What does the <head> element contain?
A: The <head> element in an HTML document contains metadata about the document, such as the title, character encoding, links to stylesheets, scripts, and other information that is not displayed directly on the webpage.

Q: How do you define a hyperlink in HTML?
A: A hyperlink in HTML is defined using the <a> (anchor) tag. The href attribute specifies the URL of the page the link goes to. For example, <a href="https://example.com">Visit Example</a> creates a link to "https://example.com."

Q: What is the role of the <title> tag in HTML?
A: The <title> tag is used to specify the title of an HTML document. This title is displayed in the browser's title bar or tab and is important for search engine optimization and user navigation.

Q: How can you insert an image into a webpage?
A: You can insert an image into a webpage using the <img> tag. The src attribute specifies the path to the image file, and the alt attribute provides alternative text if the image cannot be displayed. For example, <img src="image.jpg" alt="Description of image">.

Q: What is the difference between <div> and <span> tags?
A: The <div> tag is a block-level element used to group and style larger sections of content, while the <span> tag is an inline element used to apply styles to small sections of text or other inline elements. <div> creates a new line, whereas <span> does not.

Q: How do you create a table in HTML?
A: To create a table in HTML, use the <table> element. Inside it, use <tr> to define rows, <th> for table headers, and <td> for table data cells. For example, <table><tr><th>Header</th></tr><tr><td>Data</td></tr></table>.

Q: What is the purpose of the <form> element?
A: The <form> element is used to create interactive forms that allow users to input data, which can then be submitted to a server. Forms can include various input types such as text fields, checkboxes, and buttons.

Q: How do you create an unordered list in HTML?
A: To create an unordered list in HTML, use the <ul> element to define the list, and the <li> element to define each list item. For example, <ul><li>Item 1</li><li>Item 2</li></ul> creates a bulleted list.

Q: What is the function of the <meta> tag?
A: The <meta> tag provides metadata about the HTML document, such as character encoding, author, description, and keywords. It is placed within the <head> element and helps search engines and browsers understand the content of the page.

Q: How can you include a CSS stylesheet in an HTML document?
A: To include a CSS stylesheet in an HTML document, use the <link> tag within the <head> element. The rel attribute specifies the relationship as "stylesheet," and the href attribute provides the path to the CSS file. For example, <link rel="stylesheet" href="styles.css">.

Q: What does the alt attribute do in an <img> tag?
A: The alt attribute in an <img> tag provides alternative text for an image if it cannot be displayed. This text is used by screen readers for accessibility and appears if the image fails to load.

Q: How do you make a list item appear with a number?
A: To create a numbered list, use the <ol> (ordered list) tag, which automatically numbers the list items. Each item is defined using the <li> (list item) tag. For example, <ol><li>First</li><li>Second</li></ol> creates a numbered list.

Q: What is the <link> tag used for in HTML?
A: The <link> tag is used to link external resources to the HTML document, such as stylesheets or icons. It is placed within the <head> section and is commonly used to include CSS files.

Q: How do you create a text area in a form?
A: To create a text area in a form, use the <textarea> tag. It allows users to input multiple lines of text. The rows and cols attributes can be used to define the size of the text area. For example, <textarea rows="4" cols="50"></textarea>.

Q: What does the target attribute do in a link?
A: The target attribute in an <a> tag specifies where to open the linked document. For example, target="_blank" opens the link in a new tab or window, while target="_self" opens it in the same frame or window.

Q: How can you create a radio button in HTML?
A: To create a radio button in HTML, use the <input> tag with the type attribute set to "radio." Each radio button should have the same name attribute to group them together, allowing only one to be selected at a time. For example, <input type="radio" name="group1" value="Option1">.

Q: What is the purpose of the <script> tag in HTML?
A: The <script> tag is used to include and execute JavaScript code in an HTML document. It can be placed in the <head> or <body> sections and can reference external JavaScript files using the src attribute or include inline JavaScript code.

Q: How do you specify the language of an HTML document?
A: The language of an HTML document is specified using the lang attribute in the <html> tag. For example, <html lang="en"> indicates that the document is in English.

Q: What is the href attribute used for?
A: The href attribute in an <a> tag specifies the URL of the page the link goes to. It stands for "hypertext reference" and defines the destination of the hyperlink.

Q: How do you create a dropdown menu in HTML?

Q: What is the action attribute in a form?
A: The action attribute in a <form> tag specifies the URL where the form data should be sent when the form is submitted. It defines the destination for the form's data.

Q: How do you define a table header in HTML?
A: A table header in HTML is defined using the <th> tag within a <tr> (table row) tag. The <th> tag is used to represent header cells, which are bold and centered by default. For example, <tr><th>Header 1</th></tr>.

Q: What does the <meta charset="UTF-8"> tag do?
A: The <meta charset="UTF-8"> tag specifies the character encoding for the HTML document. UTF-8 is a character encoding capable of encoding all possible characters and is widely used for web documents.

Q: How do you add comments in HTML?

Q: What is the purpose of the <footer> tag in HTML?

Q: How do you use the id attribute in HTML?
